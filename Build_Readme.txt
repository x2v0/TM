Проект содержит:
- библиотеку для работы с удалённым сервером обработки плана облучения TMClient.dll
- примеры использования библиотеки TMClient.dll расположенные в директории .\Test\

Для сборки библиотеки TMClient.dll необходимо выполнить pbuild.bat
(запускается по двойному щелчку мышкой).
После окончания сборки, TMClient.dll будут лежать в директории .\Redist\
