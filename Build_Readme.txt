Проект содержит:
- библиотеку для работы с удалённым сервером обработки плана облучения TM.dll
- примеры использования библиотеки TM.dll расположенные в директории .\Test\

Для сборки библиотеки TM.dll необходимо выполнить pbuild.bat build.ps1
(запускается по двойному щелчку мышкой).
После окончания сборки, TMClient.dll будут лежать в директории .\Redist\
